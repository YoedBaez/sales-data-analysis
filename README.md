# Retail Sales Performance Analysis

## Overview

This project analyzes retail sales data using Microsoft Excel PivotTables to identify top-performing product categories, geographic sales trends, and seasonal revenue patterns.

## Dataset

* Superstore retail sales dataset
* Approximately 10,000 transactions

## Tools Used

* Microsoft Excel
* PivotTables
* GitHub

## Key Findings

### Top Product Category

Technology generated the highest sales revenue with approximately $836,154.

### Top Revenue State

California generated the highest sales revenue with approximately $457,688.

### Top Revenue Month

November produced the highest monthly sales revenue, indicating strong seasonal demand.
